# Duplicate Detector

## Overview

A simple system to detect duplicate or similar text entries.

---

## Features

* Text similarity calculation
* Duplicate detection
* Threshold-based matching

---

## Files

* `similarity.py` → calculates similarity
* `detect.py` → finds duplicates

---

## Usage

```bash id="p2x7vk"
python detect.py
```

---

## Example Output

```id="n5k8zt"
('python is great', 'python is awesome', 0.66)
```

---

## Use Case

* Plagiarism detection
* Data cleaning
* Content similarity analysis
